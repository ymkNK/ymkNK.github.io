安装atom的atom-beautify

Dec 07, 2018
阅读时长: 1 分钟

前言

想给atom安装beautify的插件 使用apm install atom-beautify 和是用preference中的插件安装都失败了

Start

使用github 克隆到本地，然后安装

 cd ~/.atom/packages
 git clone https://github.com/Glavin001/atom-beautify.git
 cd atom-beautify
 npm install

Success

使用control + alt + b 组合键 即可
